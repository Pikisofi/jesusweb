# ⛪ Las Fiestas de la Iglesia Católica en España 🇪🇸

¡Bienvenidos al repositorio de nuestro proyecto escolar de Religión! Este sitio web ha sido diseñado y desarrollado con muchísimo cariño por **Eva V.L.** y **Sofía V.L.**, alumnas de **6.º de Primaria**.

El objetivo de esta página es dar a conocer las celebraciones litúrgicas más importantes de la Iglesia Católica en España de una forma interactiva, alegre, amigable y muy fácil de entender para niños de nuestra edad.

---

## 🎨 Características del Diseño y Estructura

* **Estética Premium y Suave:** Diseñado con una paleta de colores litúrgicos pastel (morado, oro, verde, cielo) sobre un cálido fondo crema, usando la tipografía redondeada `Fredoka` que da un aspecto súper amigable e infantil.
* **Explorador Interactivo:** Un panel central con tarjetas interactivas de las 5 celebraciones clave (Navidad, Cuaresma, Semana Santa, Pascua y Santiago Apóstol). Al hacer clic en *"Descubrir más"*, se abre una ventana emergente muy limpia con:
  * Explicación sencilla de qué se celebra.
  * Tradiciones tradicionales de España (como las torrijas, la cabalgata o la Mona de Pascua).
  * **💡 ¿Sabías qué?:** Una sección especial de datos curiosos para aprender de forma divertida.
* **Sección de Autoras:** Un espacio especial que nos presenta como estudiantes y compañeras de 6.º de Primaria.

---

## 🛠️ Tecnologías Utilizadas

Para garantizar que la web cargue instantáneamente y sea compatible con cualquier dispositivo, hemos utilizado:
1. **HTML5:** La estructura base de toda la página.
2. **Tailwind CSS:** Para diseñar el estilo minimalista, las sombras y los hermosos colores.
3. **Vanilla JavaScript:** El "motor" de interactividad que maneja la apertura y cierre de las tarjetas de forma rápida y sin recargar la página.

## 🚀 Cómo Publicar la Web en GitHub Pages (¡En 3 Pasos!)

Para que tus compañeros y profesores puedan ver tu página web en internet con un enlace público de forma totalmente gratis, sigue estos sencillos pasos:

### **Paso 1: Guardar y subir el código a tu GitHub**
Haz un *commit* y sube los archivos de este directorio a tu repositorio en GitHub:
1. Abre tu terminal en esta carpeta.
2. Escribe los siguientes comandos:
   ```bash
   git add .
   git commit -m "feat: estructura inicial de la web escolar"
   git push -u origin main
   ```
   *(Si el repositorio remoto te pide rama principal, asegúrate de que sea `main` o `master`).*

### **Paso 2: Activar GitHub Pages en el repositorio**
1. Entra a tu repositorio en la web de GitHub: `https://github.com/Pikisofi/jesusweb`.
2. Haz clic en la pestaña **Settings** (Configuración ⚙️) en el menú superior.
3. En la barra lateral izquierda, busca el menú **Pages** (bajo la sección *Code and automation*).
4. En el apartado **Build and deployment**, busca la opción **Branch** (Rama):
   * Cambia el selector de *None* a **`main`** (o la rama que hayas subido).
   * Deja la carpeta en **`/ (root)`** (raíz).
   * Haz clic en **Save** (Guardar 💾).

### **Paso 3: ¡Listo para compartir! 🎉**
* Espera de 1 a 2 minutos.
* Al recargar la página de configuración, verás un cartel arriba del todo diciendo:
  > *"Your site is live at https://pikisofi.github.io/jesusweb/"*
* ¡Haz clic en el enlace para ver tu obra de arte y compártelo con tus profesores de clase!
